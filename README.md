## ✨ Description
**MonoBlitz** is a fast-paced 2D top-down arcade shooter built in Unity. Step into a minimal sci-fi arena where your reflexes and accuracy are tested against an onslaught of enemies. Use your mouse to aim and shoot, while dodging enemies and maintaining your health bar. With a sleek UI, custom crosshair, responsive controls, and satisfying SFX, MonoBlitz delivers a pure shooting challenge.

How long can you survive the swarm?

---

## 🔥 Key Features
- 🎮 **Smooth Player Movement**: WASD to move, mouse to aim and shoot.
- 🎯 **Custom Crosshair Cursor**: White crosshair improves visibility on a dark arena.
- ⚠️ **3 Enemy Types**:
  - 🟥 **Follow Enemy** – Chases the player and explodes on contact.
  - 🟦 **Shooting Enemy** – Stops at mid-range and fires projectiles at you.
  - 🟨 **Laser Enemy** – Locks into range and fires a steady damaging laser beam.
- 🧠 **Adaptive AI Behavior**: Each enemy reacts based on proximity and aggression logic.
- 💥 **Bullet Spread System**: Adds realism and challenge to shooting.
- ❤️ **Health Bar UI**: Displays real-time player health visually.
- 💯 **Score System**: Earn points for every enemy defeated with animated popup feedback.
- ⏸️ **Pause Menu**: Resume, Restart, or Quit the game any time during gameplay.
- 🔊 **Sound Effects**: Satisfying audio feedback for shooting, damage, and death.

---

## 🕹️ How to Play
- **Move**: `W`, `A`, `S`, `D` or Arrow Keys  
- **Aim**: Move the mouse cursor  
- **Shoot**: Left Mouse Button

## ⚙️ Tech Stack
- **Engine**: Unity 2022.3.x
- **Language**: C#, .Net
- **UI Framework**: TextMeshPro + Unity UI Toolkit
- **Audio**: Unity AudioSource, Custom Audio
- **Object Pooling**: For efficient bullet and enemy instantiation
- **Input Handling**: Unity New Input System

---

## 🔧 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Rachit-Dwivedi-R-D/MonoBlitz.git
   ```
2. Open the Unity Editor.
3. In Unity, go to File > Open Project and select the Sharp Shooter folder.
4. Once the project loads, go to File > Build Settings.
5. Select your target platform (e.g., Windows, Mac, or WebGL).
6. Click Add Open Scenes to ensure all necessary scenes are included in the build.
7. Click Build and choose a directory to save the executable.
8. Run the built executable file to play the game.
